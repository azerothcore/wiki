# Herramientas útiles

Estas son las herramientas más útiles para el desarrollo de AzerothCore.

- [Keira3](https://www.azerothcore.org/Keira3/) es un editor de base de datos para AzerothCore. Aprende más sobre la herramienta [aquí](https://github.com/azerothcore/keira3#readme).

- [SpellWork](https://github.com/azerothcore/spellwork) es una herramienta para explorar y mostrar información sobre los hechizos de World of Warcraft. Aprende más sobre la herramienta [aquí](https://github.com/azerothcore/spellwork#readme).

- [Unused Guid Search](https://github.com/azerothcore/Unused-Guid-Search) es una herramienta para encontrar huecos de entry o GUID en la base de datos. Aprende más sobre la herramienta [aquí](https://github.com/azerothcore/Unused-Guid-Search#readme).

- [AzerothCore Flags Checker](https://www.azerothcore.org/flag-checker/) es una herramienta web para encontrar todos los flags de una máscara de bits para creatures. Aprende más sobre la herramienta [aquí](https://github.com/azerothcore/flag-checker#readme).

- [WoW Database Editor](https://github.com/BAndysc/WoWDatabaseEditor) es un IDE para Smart Scripts y edición de base de datos. Aprende más sobre la herramienta [aquí](https://github.com/BAndysc/WoWDatabaseEditor#readme)

- [AoWoW](https://github.com/azerothcore/aowow) es un visor de armory/base de datos adaptado para AzerothCore. Aprende más sobre la herramienta [aquí](https://github.com/azerothcore/aowow#readme)

Si quieres encontrar más herramientas, echa un vistazo al [Catálogo de AzerothCore](https://www.azerothcore.org/catalogue.html).

Si has creado una herramienta útil que te gustaría que fuera forkeada o destacada en el catálogo, únete al servidor de Discord y menciona a alguien del Staff.
