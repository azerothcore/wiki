### Checklist personal

- ¿Estoy usando la última versión/revisión de AzerothCore? ¿Necesitas actualizar y probar de nuevo antes de reportar?

- ¿Estoy usando alguna modificación personalizada de la base de datos o del core que pueda causar el problema?

- ¿Estoy usando algún módulo que pueda causar el problema?

- ¿Incluí en qué **revisión/hash/commit** estoy?

- ¿Incluí qué **versión** de base de datos estoy usando?

- ¿Incluí qué sistema operativo estoy usando?

- ¿Incluí los pasos para reproducir el problema?

- ¿Expliqué qué está mal y cuál es el comportamiento esperado?

- ¿Reporté solo un bug por issue?
